# Fast Fourier Transform (2022)

#### ――――――――――

- *Code Editor* : VSCode.
- *Language* : C.
- *Vectorization Extension* : AVX2.

#### ――――――――――

Projet d'implémentation HPC (High Performance Computing) de la multiplication efficace de polynômes à coefficients dans Z/pZ en utilisant l'algorithme FFT (Fast Fourier Transform) et avec la vectorisation (AVX2).

#### ――――――――――

Ce projet a été fait en binôme dans le cadre du cours LU2IN013 (Projet de Développement) à Sorbonne Université.
